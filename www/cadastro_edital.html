✦ Etapa 3 de 4
Importar edital
Cole o JSON gerado ou adicione as matérias manualmente. Atenção: Este processo automatizado pode apresentar inconsistências. Verifique se todos os tópicos e subtópicos foram incluídos corretamente. Ocasionalmente, a IA pode gerar redundâncias ou tópicos desnecessários. Você poderá realizar ajustes, inclusões ou exclusões após a importação.
🤖 Prompt para gerar o JSON (testado no Gemini)
Converta o edital abaixo para JSON seguindo exatamente este modelo: { "materias": [ { "id": "mat_001", "nome": "Nome da Matéria", "topicos": [ { "id": "top_001", "nome": "Nome do Tópico", "subtopicos": [ { "id": "sub_001", "nome": "Nome do Subtópico" } ] } ] } ] } Mantenha a hierarquia matéria → tópico → subtópico. Quando um tópico contiver assuntos distintos separados por vírgula, "e" ou ";", separe-os em subtópicos individuais. Os IDs devem seguir o padrão mat_001, top_001, sub_001 incrementando. Não adicione campos extras. Retorne apenas o JSON, sem explicações. [Cole o edital abaixo desta linha]
